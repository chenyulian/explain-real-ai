# AI 词汇速通

一本正经的 AI 术语学习站。

> **LLM = 溜溜梅**
> **Transformer = 跨性别前任**
> **JSON = 杰森**
> **machine unlearning = 机器不学了**

---

## 这是什么

收录 AI 领域常见英文术语，每个词条都包含：

- **权威释义**（一本正经地）
- **谐音速记**（胡说八道地）
- **学术补充**（其实也不太正经）
- **使用例句 / 视觉案例**

站内一切释义均经过严格的事实核查（并没有）。

## 灵感来源

本站灵感来自小红书用户 **[@时光史官](https://www.xiaohongshu.com/user/profile/5d806ef50000000001005a16)** 2026 年 7 月发布的笔记《人工智能常见英语词汇学习》。

> 原帖笔记 ID：`6a561753000000002100878d`。出于隐私考虑，未在 README 中贴带分享凭证的完整 URL——可通过上方作者主页登录后查看。

原帖列举了 `LLM = 溜溜梅`、`transformer = 跨性别前任`、`machine unlearning = 机器不学了` 等谐音梗，将严肃的 AI 术语与中文同音字巧妙嫁接。

本站把这种「一本正经地胡说八道」从静态笔记扩展为一个可查询、可收藏、可分享的独立站点。

> 如果原帖作者不希望本站使用其创意，请通过 [Issues](../../issues) 联系我，会立即处理。

## 技术

- 纯静态站点（HTML + CSS + JSON）
- 零依赖，零构建，双击即可在本地打开（部分浏览器对 `fetch` 有限制，建议用 `http-server`）
- 字体：Inter + JetBrains Mono
- 设计语言：OpenAI × shadcn/ui

## 本地运行

```bash
# 任选一种静态服务器
npx http-server -p 8080
# 或
python -m http.server 8080
```

打开 http://localhost:8080
